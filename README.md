# Hunt Showdown 1896 Performance Config # 
### Кастомный system.cfg сделанный с упором на максимальное повышение FPS и отключение всего лишнего в игре ### 
### Обновлено: 31.01.26 для Cryengine 5.11 ####
## Что такое `system.cfg`
`system.cfg` — это конфигурационный файл движка CryEngine с наивысшим приоритетом при запуске игры, позволяющий вручную настраивать внутренние параметры для детальной оптимизации.
Готовый cfg файл подойдет тем кто хочет повысить FPS в игре.

### ⚠️ Важно
Перед использованием обязательно сделайте резервную копию своих настроек, в случае возникновения проблем вы сможете сделать откат. 
Для этого перейдите в директорию игры: `D:\Steam\steamapps\common\Hunt Showdown 1896` и сохраните `game.cfg` и `Profiles` 

## Установка

• Скачайте файл **`system.cfg`**

• Открой папку игры: `D:\Steam\steamapps\common\Hunt Showdown 1896` и вставьте `system.cfg`

• Запустите игру

## Возврат к стандартным настройкам
• Удалите файл **`system.cfg`**  

• Удалите папку `USER` в директории игры
